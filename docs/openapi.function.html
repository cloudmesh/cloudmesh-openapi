This is a main entry point into the module. This function will generate the full OpenApi YAML formatted specification for a module with one single function.

Parameters

  • f – python function object

  • filename – python module file name

  • serverurl – server url for flask service

  • outdir – output directory where openapi spec yaml will be written to

  • yamlfile – openapi spec yaml file name

  • dataclass_list – list containing all data class objects

  • write – flag to write out openapi spec yaml to a file. True by default.

Returns

no return but if write flag is set it will write out yaml to file.

generate_openapi_class(class_name=None, class_description=None, filename=None, func_objects=None, serverurl=None, outdir=None, yamlfile=None, dataclass_list=None, all_function=False, enable_upload=False, write=True)[source]¶

This is a main entry point into the module. This function will generate the full OpenApi YAML formatted specification for a python class or module with multiple functions.

Parameters

  • class_name – python class name

  • class_description – python class description

  • filename – python module file name

  • func_objects – all function objects in input python file

  • serverurl – url for flask service

  • outdir – output directory where openapi spec yaml will be written

  • yamlfile – file name for openapi spec yaml

  • dataclass_list – list containing all data class objects

  • all_function – flag to process all functions in input python file. False by default.

  • enable_upload – flag to support file upload function. False by default.

  • write – flag to write out openapi spec yaml to a file. True by default.

Returns

no return but if write flag is set it will write out yaml to file
